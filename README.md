## Overview

The Kaden is a full-stack automation platform designed to provide secure authentication, real database integration, and a modular chatbot infrastructure.

It is built using professional software engineering principles, with clear separation between frontend, backend, and database layers.

This project represents real infrastructure — not a prototype.

---

## Core Features

- Secure user authentication system  
- Password encryption using bcrypt  
- JWT-based session management  
- PostgreSQL database integration (Supabase)  
- Modular backend architecture  
- Modern chatbot interface  
- Production-ready project structure
